# 个人数据可视化展示项目

这是一个基于Web的数据可视化展示项目，包含多个不同风格和实现方式的数据可视化页面，主要用于展示商品销售数据和商家对比分析。

## 📁 项目文件结构

```
├── index.html          # 主页面 - 使用ECharts的完整数据可视化
├── simple_viz.html     # 简化版 - 纯CSS和SVG实现的数据可视化
├── visualization.html  # 基础版 - 简化的ECharts数据可视化
└── README.md          # 项目说明文档
```

## 🎯 项目特点

### 🌟 统一的设计风格
- **配色方案**: 采用金粉色渐变主题 (#ff6b9d, #feca57)
- **背景设计**: 深色渐变背景，营造现代科技感
- **响应式布局**: 适配不同屏幕尺寸的设备
- **动画效果**: 流畅的页面加载和交互动画

### 📊 数据可视化功能
1. **商品销售额分析**: 柱状图展示各商品销售情况
2. **商家销售对比**: 折线图对比不同商家销量
3. **商品类别占比**: 饼图展示销售占比分布
4. **综合数据分布**: 散点图展示多维度数据关系

## 🔧 技术实现

### index.html (完整版)
- **框架**: [ECharts 5.4.3](https://echarts.apache.org/)
- **特点**: 
  - 完整的图表交互功能
  - 星空背景动画效果
  - 统计数据卡片展示
  - 响应式图表布局
- **文件大小**: 22.91 KB

### simple_viz.html (CSS/SVG版)
- **技术**: 纯CSS3 + SVG动画
- **特点**:
  - 无外部依赖
  - 自定义柱状图、折线图、饼图
  - CSS动画效果
  - 轻量级实现
- **文件大小**: 16.84 KB

### visualization.html (基础版)
- **框架**: ECharts 5.4.3
- **特点**:
  - 简化的图表配置
  - 加载动画和错误处理
  - 基础的交互功能
  - 移动端优化
- **文件大小**: 14.9 KB

## 🚀 使用说明

### 本地运行
1. 下载项目文件到本地
2. 使用浏览器打开任意HTML文件即可查看
3. 推荐使用Chrome、Firefox、Edge等现代浏览器

### 部署到服务器
1. 将所有文件上传到Web服务器
2. 确保服务器支持HTTPS（ECharts CDN需要）
3. 访问对应的HTML文件即可

### 自定义配置
- **数据修改**: 在JavaScript部分修改对应的data数组
- **配色调整**: 修改CSS中的颜色变量
- **图表类型**: 参考ECharts文档更换其他图表类型
